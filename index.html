Häufige Fragen

Was Betriebe oft fragen

Die wichtigsten Punkte vorab, und wenn etwas offen bleibt, schreiben Sie mir einfach.

Ist die Einrichtung kompliziert für mich?

Nein. Ich richte alles für Sie ein und passe es an Ihren Betrieb an. Sie müssen nichts installieren und brauchen kein technisches Wissen.

Brauche ich neue Software auf meinem Computer?

Nein. Sie müssen nichts installieren. Das System läuft komplett auf Googles sicheren Servern, Sie bedienen alles bequem im Browser oder per Handy-App.

Bekommt der Kunde wirklich sofort eine Antwort?

Ja. Sobald eine Anfrage eingeht, verschickt das System in Sekunden eine automatische Eingangsbestätigung mit Ihrem Namen, auch nachts und am Wochenende. Den richtigen Terminvorschlag schicken dann Sie mit einem Klick.

Was ist mit dem Datenschutz (DSGVO)?

Ihre Anfragedaten gehören Ihnen. Ich verarbeite sie nur in Ihrem Auftrag, dafür schließen wir einen Auftragsverarbeitungsvertrag (AVV nach Art. 28 DSGVO). Die Daten jedes Betriebs werden getrennt gespeichert, und Sie erhalten sie jederzeit als Export. Das Anfrageformular enthält einen Einwilligungstext zur Datenverarbeitung.

Kann ich monatlich kündigen?

Ja. Es gibt keine lange Vertragsbindung, der laufende Betrieb ist monatlich kündbar.

Was passiert mit meinen Daten, wenn ich kündige?

Ihre Daten gehören Ihnen. Bei einer Kündigung erhalten Sie alle Anfragen und Kontakte als Export (z. B. als Excel- oder CSV-Datei) und können nahtlos weiterarbeiten. So bleiben Sie jederzeit unabhängig und sind nicht an mich gebunden.

Wie schnell bin ich startklar?

In der Regel innerhalb von wenigen Tagen (meist 3 bis 5 Werktage). Sie geben mir die wichtigsten Infos zu Ihrem Betrieb, um den Rest kümmere ich mich.

Kann ich später weitere Bausteine dazunehmen?

Ja. Das System ist ein Baukasten. Sie starten mit dem Wesentlichen und schalten weitere Module, etwa Live-Dashboard, Preisrechner oder den Wartungsautopilot, dann frei, wenn Ihr Betrieb sie braucht.

Gibt es versteckte Kosten?

Nein. Einmalige Einrichtung plus fester Monatspreis je nach Paket, keine versteckten Gebühren.

Lohnt sich das schon bei wenigen Anfragen im Monat?

Gerade dann. Wenn Sie nur wenige Anfragen bekommen, ist jede einzelne besonders wertvoll, eine verlorene tut richtig weh. Schon eine gerettete Anfrage (oft 800 bis 3.000 € Auftragswert) bezahlt das System für viele Monate.
