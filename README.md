# Anfrage-System — Verkaufs-Website

Statische Verkaufs-Website fuer das **KI-Anfrage-System fuer Handwerksbetriebe**
von Florian Boldt.

Jede Kundenanfrage wird automatisch erfasst, sortiert und beantwortet,
Terminvorschlag oder Absage gehen automatisch raus.

## Inhalt

- `index.html` — Landingpage (Problem, Produkt, Preise, FAQ, Kontakt)
- `impressum.html` — Impressum (Pflicht in DE)
- `datenschutz.html` — Datenschutzerklaerung (Pflicht in DE)

## Veroeffentlichung

Wird kostenlos ueber **GitHub Pages** gehostet. Kein Build noetig, reines HTML/CSS,
einfach im Browser oeffnen.
